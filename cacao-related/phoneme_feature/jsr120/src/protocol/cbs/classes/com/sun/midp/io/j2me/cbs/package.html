This is the optimized implementation for WMA 2.0, which provides classes that allow Java applications to access CBS functionality on a mobile device.

CBS Implementation

The CBS has its own Protocol implementation that calls the native platform's CBS functionality.

  • The CBS URL connection string does not support a designated host.
  • The CBS Connector.open() will throw an IOException if an attempt is made to open for WRITE, since it is an inbound-only protocol.
  • Attempts to call send will throw an IOException.

Sending Messages to a CBS Port

CBS support is only available for inbound messages. The rules for fragmenting large messages and the basic payload types for GSM 7-bit text, UCS-2 text, and binary messages are the same for CBS and SMS messages.
